Apa itu Domain?

Domain Name pada Internet adalah nama unik yang diberikan untuk mengidentifikasi nama server komputer seperti Web Server atau Mail Server yang terhubung ke jaringan komputer, baik intranet maupun internet.

Saat kita membeli, mendaftar, dan menggunakan nama domain untuk situs web, pengguna akan lebih mudah mengingat. Saat pengunjung memasukkan nama domain di browser mereka, komputer mereka mengirimkan permintaan ke sekelompok server yang disebut Domain Name Sistem (DNS). Kita tidak perlu benar-benar tahu apa yang terjadi di balik layar di sini. Yang perlu kita ketahui adalah bahwa DNS kemudian merespons dengan alamat IP dari server hosting situs web kita.

Anggap saja seperti ini - orang lebih mengingat nama di daftar kontak telepon daripada nomor telepon sebenarnya. Dengan cara yang sama, lebih mudah bagi orang untuk mengingat nama domain daripada alamat IP.
