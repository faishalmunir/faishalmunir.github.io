Berkenalan dengan Markdown!

November 02, 2020

Markdown?

Sebuah tools untuk melakukan konversi text-to-HTML yang bisa digunakan ketika membuat konten web. Dengan menggunakan Markdown kita dapat menulis konten HTML dengan format yang mudah dibaca dan mudah pula untuk ditulis (easy-to-read dan easy-to write).

Markdown dibuat oleh John Gruber sejak 2004 dengan bahasa Perl. Tujuan utama dari pembuatan Markdown adalah untuk mempermudah penulisan format plain text dan sebagai opsi untuk bisa mengkonversinya ke HTML dan XHTML.

Saat ini sudah banyak web-web besar yang menggunakan Markdown seperti GitHub, Reddit, Stackoverflow dan SourceForge.

Kenapa Menggunakan Markdown?

  • Markdown dapat digunakan untuk semuanya. Orang - orang menggunakannya untuk membuat website, dokumen, cacatan, buku, presentasi, pesan email, dan dokumentasi teknikal.
  • Markdown sangat portable. file yang berisi teks berformat Markdown dapat dibuka menggunakan hampir semua aplikasi.
  • Markdown adalah platform bebas. Kita dapat membuat teks berformat markdown diperangkat dan sistem operasi apapun.

Kick the Tires

Cara terbaik untuk memulai Markdown adalah dengan menggunakannya dan lebih mudah berkat berbagai alat yang gratis.

Kita bahkan tidak perlu mengunduh apapun. Ada beberapa editor Markdown Online yang dapat digunakan untuk mencoba menulis Markdown. Dillinger adalah salah satu editor Markdown online terbaik. Buka saja situsnya dan mulailah mengetik di panel kiri dan pratinjau akan muncul di panel kanan.

Bagaimana cara kerjanya?

Dillinger membuat penulisan Markdown menjadi mudah karena menyembunyikan hal-hal yang terjadi di baliknya, tetapi perlu ditelusuri bagaimana proses tersebut bekerja secara umum.

Saat kita menulis Markdown, teks disimpan dalam file teks biasa yang memiliki ekstensi .md atau .markdown. lalu apa? Bagaimana file berformat Markdown Anda diubah menjadi HTML atau dokumen siap cetak?

Jawaban singkatnya adalah kita membutuhkan Aplikasi Markdown yang mampu memproses file Markdown. Ada banyak aplikasi yang tersedia - mulai dari skrip sederhana hingga aplikasi desktop yang terlihat seperti Microsoft Word. Terlepas dari perbedaan visualnya, semua aplikasi melakukan hal yang sama. Seperti Dillinger, mereka semua mengonversi teks berformat Markdown menjadi HTML sehingga dapat ditampilkan di browser web.

Aplikasi Markdown menggunakan sesuatu yang disebut prosesor Markdown (juga biasa disebut sebagai ‘parser’ atau ‘implementation’) untuk mengambil teks berformat Markdown dan menampilkannya ke format HTML. Anda dapat melihat visual dari proses ini di bawah.

Catatan: Aplikasi dan prosesor Markdown adalah dua komponen terpisah. Agar singkatnya, kita telah menggabungkannya menjadi satu elemen ("Aplikasi Markdown") pada gambar di bawah.
